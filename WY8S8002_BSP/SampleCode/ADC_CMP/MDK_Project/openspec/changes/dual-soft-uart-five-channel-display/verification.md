# 板级验证记录

## 已完成的静态验证

- Keil C51 构建通过：`0 Error(s), 0 Warning(s)`。
- P01 不再配置为 UART0 RX，Keil 活跃编译列表已移除 `uart.c`。
- Timer2 由 P01/P06 软件 UART 独占；Timer3 由 P17 消抖和应用节拍独占。

## 待板级验证

2. 分别发送 C1/C2、C3/C4 连接位 1、2、3 的帧，确认 TM1622 第 0-3 路的激活状态及 mV/mA/mW 显示换算。

4. 在一帧接收期间对另一 UART 输入起始位，确认当前帧不受影响且忙丢失计数增加。
5. 对 P17 施加一次按下与抖动波形，确认 20 ms 后只切换一次 P13 背光电平。
